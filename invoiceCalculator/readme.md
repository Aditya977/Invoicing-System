# Invoicing Calculator

1. To run the project first run "npm install".
2. Now in terminal navigate to services folder.
3. In services, run "index.js" file using node index.js.
4. When passing filepath in cmd line arguments it should be passed like as shown in example "node index.js D:\Assignment\Invoicing-System\invoiceCalculator\usage-data.json".
5. If no file path is passed then default json will be executed.
